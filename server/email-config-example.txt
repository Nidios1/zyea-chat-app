# Email Configuration for OTP Verification
# Add these to your .env file in the server directory:

EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password

# Instructions:
# 1. For Gmail: Enable 2-factor authentication, then generate an App Password
# 2. Replace 'your-email@gmail.com' with your Gmail address
# 3. Replace 'your-app-password' with the 16-character App Password from Google
# 4. Save as .env file in the server directory
# 5. Restart the server

# Alternative email services:
# - Outlook: service: 'hotmail'
# - Yahoo: service: 'yahoo' 
# - Custom SMTP: use host, port, secure options instead of service
